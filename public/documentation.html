How to use "Get Out of Jail Free"


Our utility provides potential criminals with the information needed to pull off a heist of any magnitude within Montgomery County. To access the data, the user should first navigate to the "Map" tab. The user is then presented with a map of Montgomery County, with pop-ups showing the location of where past crimes were committed. These trends can be identified and interpreted by the user. Each pop-up marker represents how safe it is to commit a crime. You can read more on Github. Data sources:

  • Montgomery County Crime Data
  • Zip Codes
  • Zillow
